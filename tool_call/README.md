# Need of Tool in AI

## Setup

1. Install dependencies:

   ```bash
   uv sync
   ```

2. Get a Gemini API key from [Google AI Studio](https://aistudio.google.com/apikey)  
   Add it to a `.env` file:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

## Run

Open the notebook:

```bash
jupyter notebook main.ipynb
```
