# MCP TYPESCRIPT EXAMPLE

## Setup

Install dependencies:

```bash
yarn
```

## Run

General:

```bash
npx @modelcontextprotocol/inspector@latest <node-path> <repo-dir>/mcp_typescript_example/dist/index.js
```

Example:

```bash
npx @modelcontextprotocol/inspector@latest /opt/homebrew/bin/node <repo-dir>/mcp_typescript_example/dist/index.js
```
