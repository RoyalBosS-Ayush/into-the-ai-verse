# OpenAI Response API & Agentic Framework

## Setup

```bash
uv sync
```

Update the provided `.env.example` file with your API keys and rename it to `.env`.

## Run

Activate the virtual environment (if not already):

```bash
source .venv/bin/activate
```

Then run:

```bash
python function_calling.py
```

```bash
python manually_approve_mcp_request.py
```

```bash
python auto_approve_mcp_request.py
```

```bash
python mcp_with_openai_agent.py
```
